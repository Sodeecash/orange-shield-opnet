# OrangeShield

## Overview
OrangeShield is a Bitcoin Layer 1 conviction vault built on OP_NET. It allows users to time-lock BTC using smart contracts and enforce financial discipline with immutable rules.

## Why Bitcoin Layer 1
- Immutable settlement
- No admin override
- Decentralized enforcement
- Finality of rules
- Native Bitcoin security

## Problem It Solves
Human financial discipline is weak. OrangeShield provides programmable commitment mechanisms so users can:
- Prevent emotional selling
- Enforce long-term holding
- Use Bitcoin’s immutability as discipline infrastructure

## Solution
- Time-locked BTC deposits
- Optional guardian mode
- Smart contract enforcement
- No bridges or wrapped assets
- Layer 1 state management

## How It Works
1. User deposits BTC
2. User selects lock duration
3. Contract enforces rules
4. Funds remain locked
5. Automatic release after expiry

## Architecture
- OP_NET smart contracts
- Bitcoin Layer 1 settlement
- No centralized backend
- On-chain state machine

## Future Vision
Building discipline infrastructure on Bitcoin.
